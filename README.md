# zh-cns-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容彼此独立，不针对任何特定域名或具体网站。

## 项目简介

本项目旨在提供一个结构化的页面归档空间，方便浏览、引用或离线查看。所有页面均为纯前端 HTML 文件，无需后端支持。

## 目录说明

- `/pages`：存放所有独立 HTML 页面，每个文件为一个完整的页面。
- `/assets`：共享资源目录，包括样式表、脚本、图片等。
- `index.html`：仓库首页，提供页面索引与导航。

## 页面归档说明

- 每个页面均为自包含的 HTML 文件，可独立打开。
- 页面内容可能随仓库更新而调整，请以最新版本为准。
- 归档页面不保证长期稳定可用，建议根据需要自行备份。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来报告问题或提出改进建议。
- 维护者会定期审查并合并合理的贡献。
- 本仓库仅用于归档与展示，不提供任何形式的服务或支持承诺。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中直接打开 `index.html` 或 `/pages/` 下的任意页面。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后访问。
